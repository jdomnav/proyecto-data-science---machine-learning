# proyecto-data-science---machine-learning
Análisis y Predicción de Precios de Juegos en Steam:  Proyecto de Machine Learning que analiza más de 85,000 juegos de Steam para predecir precios basándose en características como:  Jugadores simultáneos (PeakCCU)  Cantidad de logros y DLCs  Reseñas positivas y negativas  Tecnologías: Python, Pandas, Scikit-learn, Matplotlib.
